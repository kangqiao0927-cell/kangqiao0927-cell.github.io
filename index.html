关于我

我是一名水资源工程方向的研究生，研究方向聚焦于灌区水资源优化配置与调度。日常工作围绕分布鲁棒优化（DRO）、模型预测控制（MPC）和随机水文模拟展开，致力于在气候变化与不确定性下为灌溉系统寻找稳健的决策方案。

技术上常用 Python 科学计算栈（NumPy、SciPy、Pandas、CVXPY）做建模与优化，用 Scikit-learn 和 TensorFlow 处理水文预测与天气生成，用 QGIS 和 ERA5 再分析数据做空间分析与制图。习惯在 Jupyter Notebook 中完成从数据下载到结果可视化的全流程。

研究之余，也在折腾一个叫 Earth Online 的副项目——用程序化生成模拟板块构造、气候、洋流与生物群落，画一些幻想风格的地图。这件事让我保持对编程和可视化的热情。

项目作品

做过的一些东西，持续更新中。

灌区优化调度 — DRO 三层决策框架

面向淠史杭灌区构建 Wasserstein-DRO + CVaR 作物结构优化 → 水权分配 → MPC 滚动灌溉调度的三层框架，在气候变化不确定性下寻求稳健用水策略。

DROCVaRMPCPythonCVXPY

水库价值估计 — Lund 对冲规则

基于 ERA5 水文数据，利用随机森林入流预测与 Lund 对冲规则，构建水库蓄水的经济价值曲线 EV(S)，量化灌区缺水的经济损失。

ERA5Random ForestLund HedgingJupyter

随机天气生成器 — ResDDPM

融合 SARIMA、跳跃扩散与残差去噪扩散概率模型（ResDDPM），生成未来气候情景（2020-2070）的日尺度天气序列，用于驱动水文模拟。
